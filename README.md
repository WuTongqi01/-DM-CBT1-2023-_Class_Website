# 📚 班级网站项目：致青春 - 23数对一班

## 🌟 项目简介

这是一个为 **23数对一班** 创建的班级官方网站，记录班级成员的青春岁月和共同成长的点点滴滴。网站以“致青春”为主题，旨在为班级留下珍贵的回忆。

## 📁 项目结构

```
-DM-CBT1-2023-_Class_Website/
│
├── HTML页面文件/
│   ├── index.html          # 自动跳转页面
│   ├── main.html           # 网站首页
│   ├── profile.html        # 班级介绍页面
│   ├── members.html        # 班级成员页面
│   ├── activities.html     # 班级活动页面
│   ├── photo-album.html    # 班级相册页面
│   └── about.html          # 关于我们页面
│
├── 样式与脚本/
│   ├── main.css            # 主样式表
│   └── main.js             # 主要JavaScript功能
│
├── 图片资源/
│   ├── image/
│   │   ├── other-image/    # 通用图片资源
│   │   ├── student-image/  # 学生照片（stuimg##.jpg格式）
│   │   └── teacher-image/  # 老师照片
│   └── (根据HTML注释，还有更多分类)
│
└── README.md               # 项目说明文档
```

## 🚀 主要功能特性

### 🎨 页面设计
- **响应式布局**：适配各种设备屏幕
- **现代UI设计**：采用卡片式布局和阴影效果
- **动态导航栏**：智能隐藏/显示，随滚动变化透明度
- **高考倒计时**：实时显示距离高考的剩余时间

### 💻 技术特点
- **纯前端实现**：HTML5 + CSS3 + JavaScript
- **无框架依赖**：原生代码编写，轻量高效
- **平滑过渡动画**：CSS3动画和过渡效果
- **智能交互**：根据用户操作动态调整界面

### 📱 页面内容
1. **首页**：班级欢迎语、高考倒计时、班级特色展示
2. **班级介绍**：班级历史、发展历程、实时时间显示
3. **班级成员**：老师、班干部、同学信息展示
4. **班级活动**：活动时间线（建设中）
5. **班级相册**：照片展示（建设中）
6. **关于我们**：网站制作说明、资源下载、班级理念

## 🛠️ 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计（Flexbox布局、网格布局、响应式设计）
- **JavaScript (ES6)** - 交互功能
- **图片资源** - 本地图片管理

## 📦 安装与使用

### 方法一：直接运行
1. 下载项目文件到本地
2. 双击 `index.html` 或 `main.html` 在浏览器中打开
3. 网站将自动加载并运行

### 方法二：部署到服务器
1. 将所有文件上传到Web服务器
2. 确保文件结构保持不变
3. 通过域名或IP地址访问

### 文件路径说明
- 所有HTML文件位于根目录
- CSS和JS文件位于根目录
- 图片资源位于 `image/` 目录下
- 学生照片命名：`stuimg##.jpg`
- 老师照片命名：`teacherimg##.jpg`

## 🔧 自定义配置

### 修改班级信息
编辑相应的HTML文件：
- `profile.html` - 修改班级历史
- `members.html` - 更新班级成员信息
- `main.html` - 修改首页内容

### 更新图片资源
1. 学生照片：放入 `image/student-image/` 目录，按 `stuimg01.jpg` 格式命名
2. 老师照片：放入 `image/teacher-image/` 目录
3. 其他图片：放入 `image/other-image/` 目录

### 修改样式
编辑 `main.css` 文件：
- 修改颜色主题：调整 `#128afa`（主色调）
- 调整布局：修改相应的CSS类
- 响应式断点：`@media` 查询部分

## 📝 开发说明

### 文件命名规范
- HTML文件：小写字母，描述性名称
- CSS类名：使用连字符（kebab-case）
- 图片文件：按功能分类存放

### 浏览器兼容性
- 支持现代浏览器（Chrome, Firefox, Safari, Edge）
- 部分效果在IE中可能无法完美显示

### 性能优化
- 图片已按需加载
- CSS和JS文件已最小化
- 使用本地资源，无需外部依赖
